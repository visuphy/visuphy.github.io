Physicify SimuLab is an open source project from the Physicify YouTube channel. It aims at creating interactive physics simulations that can run natively in your browser.

Available Simulations

Gaussian Beam Propagation

Simulate the propagation of a Gaussian laser beam through various optical elements like lenses and mirrors using ABCD matrix formalism.

Author: Hussein-Tofaili

Polarization 3D Simulation

An interactive simulation demonstrating how the polarization state of light changes as it passes through various optical elements, visualized using Jones Calculus.

Author: Hussein-Tofaili

Laser Pulse Simulator

A qualitative simulation exploring how laser pulses are formed from spectral components and how spectral phase (dispersion) affects their temporal shape.

Author: Hussein-Tofaili

Ultrafast Pulse Dispersion Analyzer

A quantitative simulation that calculates ultrafast pulse temporal shape from dispersion coefficients.

Author: Hussein-Tofaili
